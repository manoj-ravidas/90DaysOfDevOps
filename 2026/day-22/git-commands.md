# Git Commands Reference

## Setup & Config

### git --version
Checks installed Git version.

Example:
git --version

### git config
Sets username and email.

Example:
git config --global user.name "manoj-ravidas"
git config --global user.email "manoj.mb789@gmail.com"

---

# Basic Workflow

#  git status
Shows current repository status.

Example:
git status

#  git add
Adds files to staging area.

Example:
git add file.txt
git add .

# git commit
Saves staged changes.

Example:
git commit -m "Initial commit"
